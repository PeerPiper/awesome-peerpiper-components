Make your Data Changable

Want to edit edit your data? Just make it Changable!

<script>
    import Changable from './Changable.svelte';
    export let name = 'Douglas';
</script>

<Changable item={{ name }} on:change />;

This component makes the prop item contentEditable, and emits a ‘change’ event every time it changes, so that consuming parents can update their copy of the data.
